# Number of fixed columns

**\[stable\]**

Returns the number of fixed columns (non-covariate) in Pmetrics data
objects.

## Usage

``` r
getFixedColNum()
```

## Value

An integer with the number of fixed columns.

## Author

Michael Neely

## Examples

``` r
if (FALSE) { # \dontrun{
getFixedColNum()
} # }
```
