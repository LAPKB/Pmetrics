# Pmetrics data file with errors

Example data set for an NPAG/IT2B run, which has been corrupted with
errors.

## Usage

``` r
badData
```

## Format

[PM_data](https://lapkb.github.io/Pmetrics_rust/reference/PM_data.md)

## Details

Errors include missing covariate on first line for subject 1,
alphanumeric covariate for gender, and trailing dose for subject 1.
