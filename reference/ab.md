# Initial range for primary parameter values

**\[stable\]**

Define primary model parameter initial values as range. For
nonparametric, this range will be absolutely respected. For parametric,
the range serves to define the mean (midpoint) and standard deviation
(1/6 of the range) of the initial parameter value distribution.

## Usage

``` r
ab(min, max)
```

## Arguments

- min:

  Minimum value.

- max:

  Maximum value.
