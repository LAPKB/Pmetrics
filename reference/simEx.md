# Example simulator output

Example simulator output

## Usage

``` r
simEx
```

## Format

R6 [PM_sim](https://lapkb.github.io/Pmetrics_rust/reference/PM_sim.md)

## Details

This is an R6 Pmetrics
[PM_sim](https://lapkb.github.io/Pmetrics_rust/reference/PM_sim.md)
object created by created by running `$sim()` on a
[PM_result](https://lapkb.github.io/Pmetrics_rust/reference/PM_result.md)
object, e.g. `NPex$sim(include = 1:4, limits = NA, nsim = 100).`
