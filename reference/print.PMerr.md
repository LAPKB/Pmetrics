# Print Data Errors

**\[stable\]**

Print the errors in a Pmetrics data file or
[PM_data](https://lapkb.github.io/Pmetrics_rust/reference/PM_data.md)
object.

## Usage

``` r
# S3 method for class 'PMerr'
print(x, ...)
```

## Arguments

- x:

  A PMerr object made by validation in
  [PM_data](https://lapkb.github.io/Pmetrics_rust/reference/PM_data.md).

- ...:

  Not used.

## Value

A printed object.
