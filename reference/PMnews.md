# Pmetrics changelog

**\[stable\]**

See changelog for Pmetrics

## Usage

``` r
PMnews(PMversion = packageVersion("Pmetrics"))
```

## Arguments

- PMversion:

  Default is the current version, otherwise a character string with the
  starting version you wish to see up to the current, e.g. “0.21”. Use
  “all” for all versions.

## Value

The changelog for the requested version.
